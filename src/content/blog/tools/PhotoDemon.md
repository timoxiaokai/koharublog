## 1. 什么是 PhotoDemon？

PhotoDemon 是一款由 [Tanner Helland](https://github.com/tannerhelland) 开发的**开源图像处理软件**，采用 **BSD 许可证**，完全免费且无广告。软件体积仅 30MB 左右，无需安装即可使用，支持 Windows XP 到 Windows 11 全系列系统。

### 核心特色
- 🚀 **超轻量**：30MB 体积，启动速度极快
- 🎯 **免安装**：绿色软件，U 盘即插即用
- 🔓 **开源免费**：BSD 协议，可商用无限制
- 💾 **低配置**：支持 Windows XP，老电脑也能流畅运行
- 🌍 **多语言**：内置简体中文，界面友好

## 3. 核心功能详解

### 3.1 图层管理
- 支持无限图层数量
- 完整的混合模式（正片叠底、滤色、叠加等）
- 图层蒙版和透明度控制
- 智能对象功能，非破坏性编辑

### 3.2 强大的滤镜系统
内置 200+ 专业滤镜：
- **调色类**：色阶、曲线、色彩平衡、通道混合器
- **模糊类**：高斯模糊、动感模糊、径向模糊、表面模糊
- **锐化类**：USM 锐化、智能锐化、边缘锐化
- **艺术效果**：油画、水彩、素描、浮雕、像素化
- **摄影修复**：降噪、红眼修复、镜头校正、透视校正

### 3.3 批处理功能
- 支持批量调整大小、格式转换、重命名
- 可录制宏并应用到整个文件夹
- 支持多种输出格式和质量设置

### 3.4 专业工具
- **选择工具**：矩形、椭圆、套索、魔棒、快速选择
- **绘图工具**：画笔、铅笔、橡皮擦、渐变、填充
- **修复工具**：污点修复、修补工具、红眼工具
- **文字工具**：支持 TTF 字体、文字图层、样式效果

## 7. 快速上手技巧

### 7.1 基本操作
1. **打开图片**：拖拽图片到软件窗口即可
2. **调整大小**：图像 → 调整大小 → 输入尺寸
3. **保存格式**：文件 → 另存为 → 选择格式（支持 30+ 格式）

### 7.2 进阶技巧
1. **批量处理**：
   - 工具 → 批处理 → 添加文件夹
   - 选择操作（调整大小、转换格式等）
   - 设置输出文件夹，开始处理

2. **宏录制**：
   - 工具 → 宏 → 开始录制
   - 执行需要录制的操作
   - 停止录制并保存宏文件

3. **图层操作**：
   - 图层 → 新建图层
   - 调整图层混合模式和不透明度
   - 使用图层蒙版进行非破坏性编辑

## 8. 下载与安装

### 官方下载地址
- **最新版本**：[PhotoDemon 官网](https://photodemon.org/)
- **GitHub 仓库**：[https://github.com/tannerhelland/PhotoDemon](https://github.com/tannerhelland/PhotoDemon)

### 系统要求
- **操作系统**：Windows XP SP3 或更高版本
- **内存**：256MB 以上（推荐 1GB）
- **硬盘空间**：50MB 可用空间
- **显示器**：1024×768 分辨率或更高

### 安装步骤
1. 下载 ZIP 压缩包
2. 解压到任意文件夹
3. 运行 `PhotoDemon.exe` 即可使用

## 9. 常见问题解答

**Q: PhotoDemon 真的完全免费吗？**
A: 是的，基于 BSD 许可证，个人和商业使用都完全免费。

**Q: 支持 PSD 文件吗？**
A: 支持 PSD 文件的导入和导出，但复杂的图层效果可能无法完全兼容。

**Q: 可以在 Mac 上使用吗？**
A: 目前仅支持 Windows 系统，Mac 用户可以考虑使用虚拟机或 Photopea 在线版。

**Q: 有手机版吗？**
A: 没有移动端，开发者专注于桌面版的优化。

**Q: 如何贡献代码？**
A: 项目完全开源，欢迎到 GitHub 提交 Issue 或 Pull Request。
