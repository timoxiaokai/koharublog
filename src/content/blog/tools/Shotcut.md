## 1. 什么是 Shotcut？

Shotcut 是一款由 **MLT Framework** 与 **FFmpeg** 驱动的开源非线性视频编辑器，采用 **GPLv3** 协议，完全免费、无广告、无功能阉割。支持 Windows、macOS、Linux 全平台，原生 64 位，界面基于 Qt 6，支持 10bit/12bit 色深、4K/8K、HDR、RAW、WebP、WebM、AV1 等前沿格式。

## 3. 安装 & 初体验

### 3.1 一键安装包
- Windows：提供 `.exe` / `.zip` 绿色版，解压即用
- macOS：Homebrew `brew install --cask shotcut`
- Linux：Flathub `flatpak install flathub org.shotcut.Shotcut`

### 3.2 首次启动向导
1. 选择主题：暗色 / 亮色 / 跟随系统
2. 自动检测硬件：OpenGL / Vulkan / Metal
3. 代理剪辑：4K 素材自动转 720p 代理，流畅预览

## 4. 核心功能深度解析

### 4.1 时间线
- 无限轨道（视频 / 音频 / 字幕）
- 复合片段（Nested Timeline），支持子项目嵌套
- 快捷键 `S` 快速分割、`R` 选择剩余、`Ctrl+Shift+A` 全选右侧

### 4.2 滤镜系统
Shotcut 把滤镜拆成 **视频 / 音频 / 文本** 三大类，共 300+ 模块，支持：
- LUT 3D（.cube）实时调色
- 关键帧动画：任意参数均可打关键帧，曲线编辑器支持贝塞尔
- 自定义 GLSL 着色器：把 Shadertoy 代码粘进去就能跑

### 4.3 导出面板
基于 FFmpeg，预设覆盖：
- 通用：H.264 / H.265 / VP9 / AV1
- 专业：ProRes 422HQ / DNxHR / FFV1 无损
- 社交平台：B 站 4K HDR、抖音 1080p 竖屏、YouTube Shorts
- 高级：可调 GOP、VBV、色域、HDR10+ 元数据

## 8. 常见坑 & 解决方案

| 问题    | 现象           | 解决                           |
|-------|--------------|------------------------------|
| 启动黑屏  | OpenGL 版本过低  | 设置 → 显示 → 改为 DirectX / Metal |
| 4K 卡顿 | 无代理          | 设置 → 代理 → 自动生成 720p          |
| 字幕乱码  | ASS/SSA 字体缺失 | 把字体文件同目录打包或转 UTF-8           |
| 导出闪退  | 含特殊字符路径      | 输出路径改为纯英文                    |
| 音画不同步 | VFR 素材       | 先「转换」→「可变帧率」→「固定 60fps」      |

## 10. 快速上手 3 个技巧

1. **智能卡点**  
   音频 → 右键 →「节奏检测」→ 自动生成标记 → 素材对齐标记，剪 Vlog 只需 3 分钟
2. **LUT 调色批量应用**  
   选中全部片段 → 滤镜 → 添加「LUT 3D」→ 勾选「应用到所有选段」，一键统一色调
3. **语音转字幕（离线）**  
   菜单 → 工具 → 「语音转文本」→ 选择 Whisper-base → 自动生成 `.srt` → 双击可编辑时间轴

## 11. 结语

Shotcut 不是「最强」剪辑软件，却可能是「最自由」的那一个：没有订阅陷阱、没有导出限制、没有平台壁垒。2026 年，它把 AI 分镜、深度抠像、离线语音字幕这些「高端功能」统统开源，让「零预算」不再等于「低质量」。如果你受够了付费激活、硬件加密狗，又想体验接近专业级的剪辑流程，不妨给 Shotcut 留 30 分钟——它或许不会让你惊艳到尖叫，却足以让你忘记「盗版」这个词。
